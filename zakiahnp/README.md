### Prerequisite

- [NodeJS](https://nodejs.org/en/)

- NPM


### How to run?

Change directory to zakiahnp

```shell
cd ~/inery-testnet-faucet-tasks/zakiahnp
```

Create .env and edit .env file with your information

```shell
nano .env
```

Install dependencies

```shell
npm install
```

Run the script

```
npm run solution
```
