### Prerequisite

- [NodeJS](https://nodejs.org/en/)

- NPM



### How to run

Change directory to ```arraylast```

```shell
cd ./arraylast
```

Edit example.env & Rename
PRIVATE_KEY=YOUR PRIVATE KEY
INERY_ACCOUNT=YOUR INERY ACCOUNT

```shell
cp example.env .env
nano .env
```

Install dependencies

```shell
npm install
```

Run the script

```
npm run solution
```
