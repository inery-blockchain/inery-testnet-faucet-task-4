### Precondition

- [NodeJS](https://nodejs.org/en/)

- NPM



### Run for linux (ubuntu)

Change directory to directory zaraki

```shell
cd ./zaraki
```

Create .env and edit the variable
PRIVATE_KEY=YOUR PRIVATE KEY
INERY_ACCOUNT=YOUR INERY ACCOUNT

```shell
nano .env
```

Install dependencies

```shell
npm install
```
