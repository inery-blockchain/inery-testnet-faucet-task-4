### Prerequisite

- [NodeJS](https://nodejs.org/en/)

- NPM

### How to run

Change directory to directory geniaweh
```shell
cd inery-testnet-faucet-tasks/geniaweh
```

Copy .env-sample

```
cp .env-sample .env
```

Edit .env file

```
nano .env
```

Install dependencies

```shell
npm install
```

Run the script

```
npm run solution
```
