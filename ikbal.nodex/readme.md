
### Run system 

Change directory to ```CHANGE_WITH_YOUR_NAME```

```shell
cd ./CHANGE_WITH_YOUR_INERY_ACCOUNT_NAME
```

Create .gitignore and edit the variable
PRIVATE_KEY=YOUR_PRIV_KEY
INERY_ACCOUNT=YOUR_NAME-_ON_DASHBOARD

### Install dependencies

```shell
npm install
```

```shell
nano .gitignore
```

### Run the script

```
npm run push
```
