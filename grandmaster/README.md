### Prerequisite

- [NodeJS](https://nodejs.org/en/)

- NPM



### How to run

Change directory to directory grandmaster

```shell
cd ~/inery-testnet-faucet-tasks/grandmaster
```


## Install dependencies

```shell
npm install
```



## Run the script :

Solution :
```
npm run solution
```

Create :
```
npm run create
```

Update :
```
npm run update
```

Delete :
```
npm run delete
```
