## Follow This Step For Running
### Build By MrPurnomo

- Run npm install
 > `npm install`
- Set your data
 > `nano .env`

## How to run CONTRACT
  **create data**
  > `npm run createCon`

  **delete data**
  > `npm run deleteCon`

  **update data**
  > `npm run updateCon`

  **read data**
  > `npm run readCon`

## How to run TOKEN
  **Create token**
  > `npm run createTok`

  **Issue token**
  > `npm run issueTok`

  **Transfer token**
  > `npm run transferTok`

  **Retire token**
  > `npm run retireTok`
