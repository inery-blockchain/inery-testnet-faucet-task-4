### Prerequisite

- [NodeJS](https://nodejs.org/en/)

- NPM

### How to run

Change directory to dezzen
```shell
cd inery-testnet-faucet-tasks/dezzen
```

Copy .env-example

```
cp .env-example .env
```

Edit .env file with your information

```
nano .env
```

Install dependencies

```shell
npm install
```



Run the script

```
npm run solution
```
