### Prerequisite

- [NodeJS](https://nodejs.org/en/)

- NPM

### How to run

Change directory to directory dhamenz
```shell
cd ~/inery-testnet-faucet-tasks/dhamenz
```

Copy .env-example

```
cp .env-example .env
```

Edit .env file with your Inery account

```
nano .env
```

Install dependencies

```shell
npm install
```



Run the script

```
npm run solution
```
