**Requirements**
Python3


**Change Directory**

```
cd ./agil
```

**Install Requirements**

```
pip install -r requirements.txt
```

**Create .env and change its value**

```
cp .env-sample .env
```

**change env variable to match with your info**

```
INERY_PK=YOUR_PRIVATE_KEY
INERY_ACC=YOUR_INERY_ACC
```

**run the script**

```
python3 ./agl.py
```
