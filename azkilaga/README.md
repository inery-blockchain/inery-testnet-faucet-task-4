### Prerequisite

- [NodeJS](https://nodejs.org/en/)

- NPM



### How to run

Change directory to ```Nama_Akun_Inery```

```shell
cd ./Nama_Akun_Inery
```


Install dependencies

```shell
npm install
```



Run the script

```
npm run solution
```
