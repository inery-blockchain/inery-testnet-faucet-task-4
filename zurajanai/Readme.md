### Things needed  ###
1. Install nodejs and npm
curl -fsSL https://deb.nodesource.com/setup_19.x | sudo -E bash - &&\
sudo apt-get install nodejs -y

2. install curl
apt install curl

### How To Run ###

1. cd ./zurajanai

2. npm install

3. cp .env-example .env

~ edit .env
nano .env

4. npm run solution

done
