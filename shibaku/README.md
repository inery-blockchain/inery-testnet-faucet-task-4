## How to run
- Run npm install
 > `npm install`
- Copy .env.example and set environment
 > `cp .env-sample .env`
- set your information
 > `nano .env`
- Command to run TOKEN
  **For Create token**
  > `npm run create`
  **For Issue token**
  > `npm run issue`
  **For Transfer token**
  > `npm run transfer`
  **For Retire token**
  > `npm run retire`

- Command to run CONTRACT
  **For Create data**
  > `npm run create`
  **For delete data**
  > `npm run delete`
  **For update data**
  > `npm run update`
  **For read data**
  > `npm run read`
