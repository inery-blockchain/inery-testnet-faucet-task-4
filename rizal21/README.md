 ### How to run

Clone repo

```shell
git clone https://github.com/rizaltjr/inery-testnet-faucet-tasks -b task4
```

Change directory rizal21

```shell
cd ~/inery-testnet-faucet-tasks/rizal21
```

Install dependencies and edit file index.mjs 

```shell
npm install
```

```shell
nano index.js 
```

Run the script

```
npm run push
```
