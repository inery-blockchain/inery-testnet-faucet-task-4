## Prerequisite

- [NodeJS](https://nodejs.org/en/)
```
sudo apt install nodejs
```
- NPM
```
sudo apt install npm
```

### How to run

Change directory to directory ttlnnt

```
cd ~/inery-testnet-faucet-tasks/ttlnnt
```


Install dependencies

```
npm install
```



Run the script

```
npm run solution
```
