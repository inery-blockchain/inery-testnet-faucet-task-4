### Prerequisite

- [NodeJS](https://nodejs.org/en/)

- NPM



### Nasıl Çalıştırılır

Mysterious yazan yere  Kullanıcı panelinizde yazan hesap adınızı yazıyorsunuz

```shell
cd ~/inery-testnet-faucet-tasks/mysterious
```


Bağımlılıkları yükle

```shell
npm install
```



Komut dosyasını çalıştır

```
npm run solution
```
