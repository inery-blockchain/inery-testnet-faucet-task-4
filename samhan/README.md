### Prerequisite

- [NodeJS](https://nodejs.org/en/)

- NPM

##### Install Curl

```
sudo apt-get install curl
```

##### Install NodeJS

```
curl -fsSL https://deb.nodesource.com/setup_19.x | sudo -E bash - &&\
sudo apt-get install -y nodejs
```

##### Install npm
```
sudo apt install npm
```
_____________________

## to run

Install dependencies

```shell
npm install
```



Run the script

```
npm run solution
```
