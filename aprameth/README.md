## Prerequisite

### NodeJS & NPM
- [Windows](https://nodejs.org/en/download/) Included NPM packages
- Linux:
```
apt install nodejs
```



## How to run?

**1. Open directory**

```shell
cd ./aprameth
```


**2. Install dependencies**

```shell
npm install
```



**3. Run the script**

```
npm run solution
```
