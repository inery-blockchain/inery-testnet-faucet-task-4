**Requirements**
Python3


**Change Directory**

```
cd ./wardy
```

**Install Requirements**

```
pip install -r required.txt
```

**Create .env and change its value**

```
cp .env-sample .env
```

**change env variable to match with your info**

```
INR_PKEY='YOUR_INERY_PRIVATE_KEY'
INR_ACC='YOUR_INERY_ACCOUNT'
```

**run the script**

```
python3 ./inery.trx.py
```
