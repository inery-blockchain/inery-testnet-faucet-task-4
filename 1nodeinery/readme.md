### first change directory to `1nodeinery`
```sh
cd 1nodeinery
```

### then Install NPM packages 
```sh
npm install 
```

### Enter your node configuration within file `node_config.js`
```sh
nano node_config.js 
```

### Do push transaction by run the `node_Push.js`
```sh
node node_Push.js
```
