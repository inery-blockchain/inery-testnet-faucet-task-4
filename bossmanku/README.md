1. Change directory to ineryjs directory

   ```
   cd ~/inery-testnet-faucet-tasks/bossmanku/ineryjs
   ```

2. Install NPM packages

   ```
   npm install
   ```

3. Copy `.env-sample` and rename it to `.env`

   ```
   cp .env-sample .env
   ```

4. Edit ```.env``` file with your information

   ```
   nano .env
   ```


## Run RPC Example

```
npm run rpc-example
```
